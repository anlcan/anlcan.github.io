Introduction

Lembas is a framework to help backend development for mobile applications. It is not a hosted backend-as-a-service solution like parse but rather a set of libraries and tools to make mobile application development faster.

Lembas is a server-entity oriented platform. You start by designing your LembasEntities, then create endpoints by defining LembasRequest and LembasResponses to interact with your entities. Once your are happy with your endpoints and entities, you generate objective-c (Java for android) code, that contains the entities as well as the request and response objects. You can iterate and update your entites for code generation takes only couple of seconds.
